## Resumo em uma frase

Voce precisa entregar uma API HTTP em `:9999`, com load balancer e pelo menos duas instancias da API, capaz de transformar cada transacao em um vetor de 14 dimensoes, consultar os 5 vetores mais parecidos em um dataset fixo de 3 milhoes de referencias e responder se a transacao foi aprovada ou negada.
