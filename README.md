# Rinha de Backend 2026 em V (vlang)

**Autor:** André · [andreluiz.is-a.dev](https://andreluiz.is-a.dev/)

> **⚠️ Experimento 100% IA:** Este projeto foi gerado inteiramente por IA em uma única sessão.

## O Contexto

Fiquei sabendo da Rinha hoje, no dia do prazo. Eu já programo em **V** há 2 anos e gosto muito da linguagem, então fiquei curioso para saber como seria o desempenho dela nesse desafio específico (busca vetorial com restrição forte de recursos).

Como não havia tempo para implementar do zero, pedi para a IA construir toda a solução (vetorização, K-Means, índice IVF, etc.) apenas para colocar o V à prova e ter um benchmark real da linguagem operando nessas condições. O objetivo principal é ver como o V se comporta, e futuramente estudar os algoritmos matemáticos gerados e reescrevê-los por conta própria.

Se a organização quiser **desconsiderar** este repositório do ranking oficial, não tem o menor problema. A submissão vale mais pelo estudo e curiosidade técnica!
